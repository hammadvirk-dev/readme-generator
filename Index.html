— Step by Step Guide —
1

Anthropic Account Banao

Sabse pehle console.anthropic.com par jao aur free account banao. Sirf email aur password chahiye — bilkul free hai.

🔗 console.anthropic.com → Sign Up → Email verify karo
2

API Key Generate Karo

Account mein login karo. Left sidebar mein "API Keys" par click karo. Phir "Create Key" dabao. Key copy karo — yeh sirf ek baar dikhti hai!

⚠️ Key kisi ke saath share mat karna — yeh tumhara secret password hai
3

Key Yahan Daalo aur Save Karo

Generator mein API Key field mein paste karo. Phir "💾 Save Key" dabao — browser mein save ho jayegi. Agli baar automatically load hogi.

✓ Save ke baad page band karke dobara kholo — key automatic load hogi
